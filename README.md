# AESOOP PROJECT
'이솝' 을 벤치마킹하여 제작한 홈페이지입니다.

## 개발환경 및 사용기술
* Front-end
  * HTML5
  * CSS3
  * Javascript 1.0
  * jQuery 1.11.0
  * Bootstrap 3.4.1
  
* Back-end
  * JAVA 11
  * JSP 2.3.3
  * Spring Framework 5.3.2
  * Eclipse IDE 4.18.0, Apache
  * tomcat 9
  * Maven

* Database
  * MySQL
  * MyBatis
  
## 기능
* 유저 페이지
  * 로그인 및 로그아웃
  * 회원가입 및 아이디, 비밀번호 찾기
  * 제품 검색, 목록, 상세페이지
  * 제품 리뷰 확인 및 작성
  * 장바구니 및 주문하기
  * 문의하기
    * FAQ 게시판 목록
    * 문의글 작성
  * 마이 페이지
    * 계정 정보 수정
    * 주문 내역 확인 및 취소
    * 리뷰 목록, 수정, 삭제
    * 회원 탈퇴
 
* 관리자 페이지
  * 회원 목록, 등록, 수정, 삭제
  * 상품 목록, 등록, 수정, 삭제
  * 주문 목록, 수정, 삭제
  * 리뷰 목록, 수정, 삭제
  * 문의글 목록
  * FAQ 목록, 등록, 수정, 삭제
